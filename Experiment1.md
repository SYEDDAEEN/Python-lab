## Experiment 1

## Aim
To write a Python program to demonstrate basic data types and operators

## Algorithm
1.Start the program.
2.Declare variables of different data types (integer, float, string, boolean).
3.Display the values and their data types.
4.Perform arithmetic operations like addition, subtraction, multiplication, and division.
5.Display the results.
6.Stop the program.

## Code
a = 10          
b = 3.5         
c = "Python"    
d = True       
print("Integer value:", a)
print("Float value:", b)
print("String value:", c)
print("Boolean value:", d)
print("Type of a:", type(a))
print("Type of b:", type(b))
print("Type of c:", type(c))
print("Type of d:", type(d))
x = 8
y = 2
print("Addition:", x + y)
print("Subtraction:", x - y)
print("Multiplication:", x * y)
print("Division:", x / y)
print("Modulus:", x % y)

## Output

Type of c: <class 'str'>  
Type of d: <class 'bool'> 
Addition: 10
Subtraction: 6
Multiplication: 16
Division: 4.0
Modulus: 0
